📄ページ数設定

推奨設定：3ページ

1ページあたり最大100件のデータが取得できます。

ページ数を設定する際の重要なポイント：

  • 3ページ設定時： 最大300件のデータを取得
  • 重複除去： 同一物件は自動削除され、実際のデータは約1/4程度
  • 最終出力： 通常50-80件程度のデータになります
⚠️ 注意事項
全駅で掲載件数が201件を下回るとエラーが発生します。統計計算のため、最低30件以上のデータが必要です。

📧メール設定

必須項目： メールアドレスは必ず入力してください。

システムがメールアドレスの入力を確認しないと動作しない仕様になっています。

有効なメールアドレス形式で入力してください（例：user@example.com）

🎯 駅選択

🐍 Python送信

Flask起動後、送信ボタンをクリック

操作手順：
  1. 路線を選択する
  2. 駅を選択する（最低3駅、最大制限なし）
  3. Python Flaskが起動していることを確認
  4. 「URL生成」ボタンをクリック
  5. 「Pythonに送信」ボタンをクリック

🔗条件別URL（上級者向け）

用途： 路線を越えた比較や特定の間取り条件での比較分析

詳細手順：
  1. SUUMOサイトにアクセス：
    https://suumo.jp/
  2. 地域・条件を選択：
    • 関東、関西などの地域を選択
    • 「賃貸」を選択
    • 都道府県を選択
    • 路線を選択
  3. 駅・表示設定：
    • 駅を一つだけ選択（前の駅が解除されていることを確認）
    • 「部屋ごとを表示」を選択
    • 右上の表示件数を100に設定
  4. 条件設定（オプション）：
    • 間取り条件
    • バストイレ別などの設備条件
  5. URL取得・編集：
    • URLをコピー
    • 末尾に「&page=」を必ず追加
    • 複数URLをコンマ区切りで入力
🚨 重要な注意事項
• すべてのURLの末尾に「&page=」を付け加えてください
• 設定ページ数×100より件数が少ない条件があるとエラーが発生
• 3ページ設定時は201件以下、4ページ設定時は301件以下でエラー
最終実行手順：
  1. 検索条件欄にURLに対応した条件名を入力
  2. Python Flask が起動していることを確認
  3. 「データ確認」ボタンをクリック
  4. 「Pythonへデータ送信」ボタンをクリック
